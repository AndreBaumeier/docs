# Linux `pygmy`

`pygmy` will start:



#### Prerequisites
1. Docker, see [the official guides](https://docs.docker.com/engine/installation/) how to do that
2. Ruby, see [the official guides ](https://www.ruby-lang.org/en/documentation/installation/)

#### Installation
    $ gem install pygmy
